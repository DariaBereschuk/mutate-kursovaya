# mutate_recovered.py — автоматический мутагенез PDB с библиотекой Данбрэка

Автономный скрипт на Python для внесения точечных аминокислотных замен в PDB-файлы. Конформация новой боковой цепи выбирается ротамерной библиотеке Данбрэка.

## Зависимости

- Python 3.8
- NumPy

## Использование

python mutate_recovered.py <pdb_file> <chain> <res_num> --to <NEW_RES> --out <output.pdb> [--lib <library_file>]

Пример: python mutate_recovered.py 1UBQ.pdb A 42 --to SER --out 1UBQ_A42_SER.pdb

## Краткое описание работы

1. Из PDB извлекаются координаты N, CA, C мутируемого остатка и его соседей.
2. Вычисляются углы φ и ψ.
3. Углы округляются до бинов библиотеки Данбрэка, выбирается наиболее вероятный ротамер (χ₁–χ₄).
4. По χ-углам и стандартной геометрии связей строится новая боковая цепь.
5. Старая боковая цепь удаляется, остов сохраняется, вставляются новые атомы.
6. Результат записывается в выходной PDB-файл.
